> Apache とは

- 世界中で使用されている web サーバーソフトウェア

- モジュール単位で機能を追加、削除可能。

> Apache の設定

- httpd.conf, または.htaccess ファイルに設定を記載。

- 基本的に大文字小文字は区別しない。

- セクションによって適用範囲を指定。

- ディレクティブによって設定値を変更。
  ServerRoot "/Applications/MAMP/Library"
  この部分がディレクティブ。
